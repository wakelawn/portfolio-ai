# sPACE Portfolio Website

A modern, responsive single-page portfolio website themed around sPACE.

## Files

- `index.html`: Main page structure and content sections.
- `styles.css`: Visual theme, responsive layout, and animations.
- `script.js`: Mobile menu toggle and dynamic footer year.

## Run Locally

Open `index.html` directly in your browser.

If you want a local server from this folder:

1. Run `python3 -m http.server 8000`
2. Open `http://localhost:8000`

## Customize

- Replace project titles and descriptions in `index.html`.
- Update colors and typography variables in `styles.css` under `:root`.
- Change contact links in the `#contact` section.
